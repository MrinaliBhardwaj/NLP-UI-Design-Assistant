UX Chatbot Project 

This project is a Python-based chatbot system designed to assist with UX/UI-related queries using structured data and text processing.

Features

- Chatbot for UX/UI queries
- PDF text extraction
- Data preprocessing
- Corpus-based response generation
- Wikipedia-based fallback (if implemented)

Technologies Used

- Python
- JSON
- NLP (basic preprocessing)
- PDF processing libraries

Project Structure
UX_CHATBOT_PROJECT/
│
├── chatbot.py # Main chatbot logic
├── blog.py # Additional processing
├── corpus.json # Dataset for chatbot responses
├── corpus.py # Corpus handling
├── preprocess.py # Text preprocessing
├── pdf_extract1.py # PDF extraction logic
├── pdf_extract2.py # Additional PDF extraction
├── wikipedia.py # Wikipedia integration
├── ux1.pdf # Sample UX document
├── ux2.pdf # Sample UX document


How to Run
1. Clone the repository:

git clone https://github.com/your-username/UX_CHATBOT_PROJECT.git

2. Navigate to project:

cd UX_CHATBOT_PROJECT

3. Run chatbot:

python chatbot.py


Example Queries

- What is UX design?
- What is usability?
- What are UX principles?
